# CROVIA Code of Conduct

CROVIA is an open-core project focused on transparency, reproducibility, and
verifiable evidence for AI data governance.  
To maintain a safe, professional, and inclusive environment, all contributors,
maintainers, auditors, operators, and community members are expected to follow
this Code of Conduct.

---

## 1. Our Standards

We commit to:

- behaving with respect, professionalism, and honesty  
- supporting newcomers, researchers, and contributors  
- providing constructive feedback  
- valuing transparency and traceability in both technical and community work  
- fostering an environment where different backgrounds and perspectives are
  welcome

Examples of unacceptable behavior include:

- harassment, discrimination, or personal attacks  
- publishing others’ private information  
- persistent disruptive behavior  
- intentionally providing misleading technical information or evidence  
- attempting to bypass or weaken CROVIA’s verification, governance,
  or security guarantees

---

## 2. Scope

This Code of Conduct applies to:

- all project spaces (GitHub repo, issues, pull requests, discussions)  
- interactions on official CROVIA channels  
- any public representation of the project or its contributors  

---

## 3. Reporting Issues

If you experience or witness a violation of this Code of Conduct, please report
it privately via email:

**conduct@croviatrust.com**

Reports will be handled confidentially.  
No retaliation of any kind will be tolerated.

---

## 4. Enforcement

Project maintainers will:

1. review all reports seriously  
2. take appropriate action, which may include:  
   - warnings  
   - temporary suspension  
   - permanent removal from project spaces  
3. document major enforcement actions (without disclosing private information)

Maintainers must recuse themselves from handling reports involving their own
conduct.

---

## 5. Attribution

This Code of Conduct is inspired by best practices from open-source projects and
adapted specifically for CROVIA’s mission:  
**verifiable, trustable, governance-grade AI data settlements.**

---

By participating in this project, you agree to abide by this Code of Conduct.
